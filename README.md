# 🛍️ VIPI - Online Shopping Landing Website

A modern, responsive landing website for VIPI - your ultimate online shopping destination.

## ✨ Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean and intuitive interface with smooth animations
- **Product Showcase**: Dynamic product grid with add-to-cart functionality
- **Hero Section**: Eye-catching banner with call-to-action button
- **Features Section**: Highlights key benefits of the platform
- **Contact Form**: Get in touch with the team
- **Shopping Cart**: Track items added to cart
- **Smooth Navigation**: Easy navigation with sticky navbar

## 📁 File Structure

```
vipi-landing/
├── index.html      # Main HTML file
├── styles.css      # Styling and responsive design
├── script.js       # JavaScript functionality
└── README.md       # This file
```

## 🚀 Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Start exploring the site!

## 🎨 Design Features

- **Color Scheme**: Modern red (#FF6B6B) and teal (#4ECDC4)
- **Typography**: Clean and readable fonts

## 🛒 Functionality

- Add products to cart
- View cart count

- Form submission
- Responsive notifications
- Mobile-friendly menu

## 📱 Responsive Breakpoints

- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: Below 768px
